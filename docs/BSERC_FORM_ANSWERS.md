## Section 4 — Project Description

### Objective * (max 500 chars)

```
Build a production-grade, open-source OSINT dashboard that fuses GDELT global events, NewsAPI, and Reddit RSS with NLP entity extraction, sentiment scoring, topic discovery, and ML forecasting to surface defense-relevant signals across 50+ countries and 5+ years of history, in real time, on a laptop.
```

*(301 / 500 chars)*

### Methodology / Approach * (max 500 chars)

```
Clean Architecture: 8-layer data pipeline (ingestion, transform, NLP, ML, tracking, registry, dashboard, deploy). Pydantic v2 schemas at every parser boundary. spaCy NER with 33 custom WEAPON patterns. DistilBERT sentiment + VADER fallback. XGBoost escalation classifier with time-aware CV. Prophet forecasting. Isolation Forest anomalies. MLflow tracking + JSON registry. Streamlit dashboard. Docker Compose stack.
```

*(415 / 500 chars)*

### Expected Outcome / Impact * (max 500 chars)

```
Full-stack dashboard with 370k+ GDELT events, 7 ingestion sources, 10 interactive Streamlit pages, 3 trained ML models (escalation horizons h1/h3/h7). 213 tests passing (208 unit + 5 e2e), 68.59% coverage, lint clean (ruff/black/isort). Multi-stage Docker image, docker-compose stack. MIT licensed, ready for Streamlit Cloud / HF Spaces / AWS deploy.
```
